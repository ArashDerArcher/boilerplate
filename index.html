Website Boilerplate

This boilerplate helps the SASS/HTML5/CSS3/JavaScript dev to simply clone this repo and get to the coding immediately

My workflow

  • Receive design PSD/examples
  • http://compass.handlino.com/ Use fire.app or compass.app
  • Clone repo
  • Watch base folder with compass or fire
  • Edit index.html & assets/sass/ files

Included

  • Selectivizr
  • respond.js
  • Modernizr
  • Formalize
  • jQuery

Tech

  • HTML5
  • CSS3 (animations included)
